# Zendesk
Zendesk Assignment
Steps to run the app:
1. Clone the repository ( git@github.com:bhanubhakta/Zendesk.git )
2. Install ruby-2.3.1 running `rvm install ruby-2.3.1`
2. run `bundle install`
3. run `ruby app.rb`
4. Then follow the instructions for search
